======================================
  arXiv Browser — Instructions
======================================

TO BUILD THE INSTALLER (.exe):
--------------------------------
1. Double-click BUILD.bat
2. Wait (5-10 minutes the first time —
   the script downloads Node.js if needed)
3. The .exe appears in the "dist\" folder
4. Share that .exe with your colleagues

COLLEAGUES NEED NOTHING ELSE:
-------------------------------
They receive the .exe, double-click it,
click "Next" a few times, and the app
is installed like any normal software.

TO UNINSTALL THE APP:
----------------------
Control Panel → Programs → arXiv Browser → Uninstall
Or: the install folder contains
"Uninstall arXiv Browser.exe"

APP FEATURES:
--------------
- Search tab: browse arXiv by category,
  keywords, date range, search field
  (All / Title / Abstract / Author)
- Library tab: save papers, add notes,
  organize in collections, export BibTeX
- AI tab (optional): enter an OpenAI or
  Anthropic API key in Settings to enable
  chat with your library
- Settings tab: AI key, GitHub sync

DATA LOCATION:
---------------
Library and settings are stored at:
C:\Users\<YourName>\AppData\Roaming\arxiv-browser\
